  In standard attention
  x = content + position          ← mixed into one vector
  Q = W_q * x,  K = W_k * x       ← computed from mixed signal
  attention = Q · K               ← contains all cross-terms, tangled

  In DeBERTa:
  Q_c = W_q_c * content            ← separate
  Q_p = W_q_p * position           ← separate
  K_c = W_k_c * content
  K_p = W_k_p * position

  attention = Q_c·K_c + Q_c·K_p + Q_p·K_c
